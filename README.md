# Courier Dash 🚴

Courier Dash is an AI-powered delivery routing game where players must deliver packages across a city grid while avoiding blocked roads.

## Features
- Grid-based delivery puzzle
- A* pathfinding AI for optimal routes
- Hint system using heuristic search
- Multiple deliveries and obstacles
- Interactive UI with animations

## AI Concepts Used
- A* Search Algorithm
- Manhattan Distance Heuristic
- Path Planning
- Optimization

## How to Play
1. Move the courier using arrow buttons.
2. Deliver all packages (📦).
3. Reach the final destination (🏁).
4. Try to minimize steps.
